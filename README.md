# How to run the automation test
1. `git clone git@github.com:dmaziar/dk_test.git`
2. `cd ./dk_test && npm i`
3. `npx playwright test`
4. To run on CI please go here https://github.com/dmaziar/dk_test/actions/workflows/playwright.yml
